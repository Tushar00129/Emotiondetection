🎯 MAIN FUNCTION: fuse_emotions()
─────────────────────────────────────────────────────────────────────────────

SIGNATURE:
    
    fuse_emotions(
        face_emotion: str,
        voice_emotion: str,
        text_sentiment: str,
        method: str = 'weighted',
        **kwargs
    ) → str


METHODS:

1. WEIGHTED (Default)
   ─────────────────────
   Weights: Face=50%, Voice=30%, Text=20%
   How: Each emotion gets score = weight × 1.0
   Best for: Simple, reliable decisions
   Usage: fuse_emotions('Happy', 'Happy', 'positive')

2. VOTING
   ───────
   How: Each modality votes; majority wins
   Best for: Democratic approach
   Usage: fuse_emotions(..., method='voting')

3. HYBRID (Most Advanced)
   ──────────────────────
   Weights: Combined with confidence scores
   How: Score = weight × confidence
   Best for: Production systems, high accuracy
   Usage: fuse_emotions(..., method='hybrid',
                        face_confidence=0.95,
                        voice_confidence=0.85,
                        text_confidence=0.70)


EXAMPLES:

# Simple weighted fusion (default)
emotion = fuse_emotions('Happy', 'Happy', 'positive')
→ 'Happy'

# Hybrid with confidences
emotion = fuse_emotions(
    'Happy', 'Angry', 'positive',
    method='hybrid',
    face_confidence=0.95,
    voice_confidence=0.60,
    text_confidence=0.80
)
→ 'Happy' (Face+Text confidence beats Voice)

# Voting method
emotion = fuse_emotions(
    'Happy', 'Happy', 'Sad',
    method='voting'
)
→ 'Happy' (2 votes vs 1 vote)
