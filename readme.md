# Multimodal Emotion Detection AI

This project detects human emotions using:
- Facial expressions
- Voice tone
- Text sentiment

The system fuses multiple modalities to predict emotions and generate AI responses.

## Features
- Face emotion detection using CNN
- Voice emotion recognition
- Text sentiment analysis
- Emotion fusion model
- AI emotional response generator

## Installation

pip install -r requirements.txt

## Run Demo

python demo/realtime_detection.py
