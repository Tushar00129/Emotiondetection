## Voice Emotion Detection - Getting Started

---

## 🚀 FASTEST START (Do This First)

```bash
# Step 1: Activate virtual environment (if not already)
.\.venv\Scripts\Activate.ps1

# Step 2: Train on YOUR voice (1 minute)
python voice_personalization.py quick

# Step 3: Run the system
python final_emotion_detection.py
```

**That's it!** The system will now use your personalized model.

---

## 📊 Test Your Progress

```bash
# See current accuracy before/after training
python voice_calibration_manager.py test
```

You'll record 5 test samples (one per emotion) and see how accurate it is.

## 🎮 During Emotion Detection

Press these keys:
- **`v`** → Record voice (1.5 seconds)
- **`a`** → Get AI response with audio
- **`r`** → Reset voice result
- **`q`** → Quit

---

## ⚠️ If Still Not Working

1. **Microphone issue?**
   - Check if microphone is connected
   - Use quiet room

2. **Still low accuracy after quick training?**
   - Try full calibration: `python voice_personalization.py full`
   - Speak more clearly when recording

3. **Want to start fresh?**
   - Delete: `voice_emotion_model_personalized.h5`
   - Delete: `voice_training_data/` folder
   - Run: `python voice_personalization.py quick`

## 💡 Pro Tips

- **Best accuracy**: Full calibration with 10 samples per emotion
- **Fastest setup**: Quick calibration (does 5 samples, takes 1 min)
- **Quality matters**: Speak naturally with clear emotion expression
- **Quiet room**: Best results in quiet environment
- **Feedback helps**: System learns from multiple calibrations
- **Test often**: Run `test` command to check progress

## 🆘 Emergency Reset

If everything goes wrong:

```bash
# Delete personalized model
del voice_emotion_model_personalized.h5 /s /q

# Delete training data
rmdir voice_training_data /s /q

# Start over
python voice_personalization.py quick
```
